Vamos aprender formatação

NEGRITO / DESTAQUE

Para colocar um texto em negrito pode se usar a tag <b> desta forma aqui (Não semântica)

Para dar destaque a um texto usamos a tag <strong> desta forma aqui (Semântica)

ITÁLICO / ÊNFASE

Para colocar um texto em Ítalico pode se utilizar a tag <i> desta forma aqui (Não semântica)

Para dar Ênfase em uma frase do texto pode se utilizar a tag <em> desta forma aqui (Semântica)

TEXTO MARCADO

Podemos criar também um texto marcado usando a tag <mark> desta forma aqui

E no outro parágrafo, temos outro texto marcado no final

TEXTO GRANDE E PEQUENO

Estamos criando um texto grande e um texto pequeno nesse parágrafo utilizando as tags <big> e <small>

TEXTO DELETADO

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado utilizando a tag <del>

TEXTO INSERIDO

Podemos marcar um como inserido para dar uma infase e indicar ele foi adicionado depois utilizando a tag <ins>

existe também o sublinhado com a tag <U> (Não semântica)

TEXTO SOBRESCRITO

Para inserir coisas do tipo X20+3 utilize a tag <sup>

TEXTO SUBSCRITO

Para inserir coisas do tipo H20 utilize a tag <sub>
